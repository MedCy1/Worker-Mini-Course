# Web Workers API - Présentation

Présentation interactive sur l'API Web Workers en JavaScript, réalisée dans le cadre du cours JS-API à l'ESGI.

## Contenu

- Qu'est-ce qu'un Web Worker ?
- Communication entre le thread principal et un worker
- Limitations et cas d'usage
- Exemple réel d'utilisation

## Utilisation

Ouvrir `index.html` dans un navigateur.  
Un exemple de démonstration est disponible dans `demo.html`.

## Navigation

- Boutons **Précédent / Suivant** en bas de l'écran
- La barre de progression indique l'avancement dans les slides
